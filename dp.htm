微軟 DP 系列認證 (Data & AI)

DP 系列專注於 Azure 雲端平台上的數據基礎建設、處理、存儲與 AI 模型開發。

DP-900: Azure Data Fundamentals

適合對象：初次接觸雲端數據服務的人員、非技術背景管理職。

準備方向：了解關聯式 (SQL) 與非關聯式數據庫、Azure 資料存儲的基本術語與服務。

DP-203: Azure Data Engineer Associate

適合對象：資料工程師、後端開發者、大數據架構師。

準備方向：深入學習 Azure Synapse, Azure Databricks, Data Lake 與 Microsoft Fabric 的整合與 ETL 實務。

DP-600: Microsoft Fabric Analytics Engineer

適合對象：現代數據平台分析工程師、熟悉 Power BI 且想跨入大數據領域者。

準備方向：這是目前最新的重點認證，需掌握 Fabric 一站式平台的數據建模、Lakehouse 與 Warehouse 的管理。
