個人專業技能實力

結合地端伺服器維護經驗與雲端數據技術，提供全方位的解決方案。

  • SQL Server Analysis Services (SSAS)

    專精於多維度數據模型 (Multidimensional) 與表格模型 (Tabular) 的建置。能夠將複雜的商業規則封裝在模型中，為企業提供高效的高速查詢與聚合分析能力。

  • SQL Server Reporting Services (SSRS)

    設計與部署標準化企業報表，包含分頁報表、鑽取報表 (Drill-through) 與自動化排程派送。確保數據能以格式化的方式精準傳達給各部門主管。

  • Power BI 視覺化與生態系

    擅長設計直觀、互動式儀表板。從數據導入、清理 (Power Query) 到發佈至 Power BI Service 的全流程管理，幫助企業實現數據驅動的決策文化。

  • DAX (Data Analysis Expressions) 開發

    具備撰寫高效 DAX 運算式的能力，處理複雜的時間智能 (Time Intelligence) 計算、動態彙總與高級過濾邏輯，優化 Power BI 模型的運算效能。

  • Python 自動化與開發

    運用 Python 進行數據自動化處理 (ETL)、自動化報表生成以及基礎的數據科學分析。熟悉 Pandas, NumPy 等庫，能與 Microsoft 技術棧深度整合。

  • Azure SQL Database 建置與管理

    具備將地端 SQL 資料庫遷移至雲端環境的實務經驗。負責 Azure SQL Database 的結構設計、效能監控、安全設定與彈性縮放，確保數據資產的安全性與高可用性。
